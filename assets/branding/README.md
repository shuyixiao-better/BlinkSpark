# BlinkSpark Branding

## Files

- `blinkspark-icon.svg`: Primary app icon (vector, editable).
- `blinkspark-logo.svg`: Horizontal brand lockup for README/release pages.
- `blinkspark.ico`: Canonical Windows app icon used by `build.rs`.
- `generated/`: Raster exports generated by script.

## Generate PNG/ICO

Run in project root:

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\generate_logo_assets.ps1
```

Generated files:

- `assets/branding/generated/blinkspark-icon-1024.png`
- `assets/branding/generated/blinkspark-icon-512.png`
- `assets/branding/generated/blinkspark-icon-256.png`
- `assets/branding/generated/blinkspark-icon-128.png`
- `assets/branding/generated/blinkspark-icon-64.png`
- `assets/branding/generated/blinkspark-icon-48.png`
- `assets/branding/generated/blinkspark-icon-32.png`
- `assets/branding/generated/blinkspark.ico`
- `assets/branding/blinkspark.ico` (copied automatically for build pipeline)

## Generate macOS ICNS

Run in project root:

```bash
./scripts/generate_macos_icon.sh
```

Generated file:

- `assets/branding/generated/blinkspark.icns`

## Design Rationale

- Eye shape: communicates blink/eye-care directly.
- Segmented circular ring: expresses timed reminder cycles.
- Spark accent: represents freshness and attention reset.
- Blue/cyan palette: calm, health-focused, desktop-friendly.
